# Terrain Tile Set (64px Pixel Art)

This folder contains 64x64 pixel-art terrain tiles requested for tactical map use.

## Included tiles

- `plain.svg`
- `road.svg`
- `forest.svg`
- `water.svg`
- `ridge.svg`
- `mountain.svg`

## Notes

- The first five match the terrain keys currently used by simulation (`plain`, `road`, `forest`, `water`, `ridge`).
- `mountain.svg` is included as an extra tile for future expansion.
