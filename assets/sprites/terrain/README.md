# Terrain Tile Set (64px Pixel Art)

This folder contains 64x64 pixel-art terrain tiles requested for tactical map use.

## Included tiles

- `plain.svg`
- `road.svg`
- `forest.svg`
- `water.svg`
- `ridge.svg`
- `mountain.svg`

## Notes

- These tiles match the active terrain keys in `TERRAIN_LIBRARY`.
- Water and ridge block ground units.
- Mountain blocks vehicles while remaining passable for infantry and air units.
