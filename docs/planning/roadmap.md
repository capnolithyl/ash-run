# Development Roadmap (Updated)

1. **Combat polish + rules clarity**
   - tighten edge cases for capture/recruit/AI order
   - improve combat readability and feedback timing
2. **Balance pass**
   - commander parity
   - unit cost/stat tuning
   - map economy pacing
3. **Objective expansion**
   - introduce non-elimination win conditions
   - mission rule variants
4. **Run rewards + meta depth**
   - post-battle reward drafting
   - unlock pacing and long-term progression hooks
5. **Content growth**
   - additional map archetypes and encounter modifiers
   - richer commander/unit interactions
6. **Presentation pass**
   - final art direction implementation
   - VFX/SFX/music iteration
7. **Stability + packaging**
   - save robustness and migration
   - release-oriented QA and platform hardening
