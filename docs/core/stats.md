# Unit Stats

- Health → Unit dies at 0
- Attack → Base damage
- Armor → Damage reduction
- Movement Range → Tiles moved
- Attack Range → Attack distance
- Stamina → Movement resource
- Ammo → Attack resource
- Luck → Random bonus effects

## Stamina

- -1 per tile moved
- 0 = cannot move

## Ammo

- -1 per attack
- 0 = cannot attack

## Luck

- Affects:
  - Bonus damage
  - Crit chance
  - Dodge chance
