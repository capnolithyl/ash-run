# Units

Ash Run '84 currently defines 12 recruitable unit types across 3 families.

## Infantry (Barracks)

- Grunt
- Breaker
- Longshot

## Vehicles (Motor Pool)

- Runner
- Bruiser
- Juggernaut
- Siege Gun
- Skyguard

## Air (Airfield)

- Gunship
- Payload
- Interceptor
- Carrier

## Notes

- Unit data includes costs, family tags, ranges, and effectiveness tags.
- Carrier is a non-attacking support-style unit in current prototype data.
