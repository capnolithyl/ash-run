# Gameplay Loop

1. Choose a commander.
2. Start a run in a save slot.
3. Buy your opening squad, then enter the next map using your current run roster.
4. Take turns moving, attacking, capturing, and holding service points.
5. Earn Intel Credits from captures and map clears while denying enemy production and income.
6. Win the map and carry surviving units forward.
7. Reset map-specific charge context, redeploy survivors without overlap, and load the next map with higher enemy pressure.
8. Lose a battle and the run ends.
9. Repeat runs to improve consistency and unlock additional commanders.
