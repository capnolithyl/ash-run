# Game Structure

## Runs

- Series of sequential battles
- Maps chosen randomly
- Difficulty increases over time
- Run ends on defeat

## Between Battles

- Player selects upgrades or rewards
- Possible rewards:
  - New units
  - Stat bonuses
  - Passive effects
  - Commander enhancements

## Meta Progression

- Currency earned after runs
- Used to unlock:
  - Commanders
  - Permanent bonuses
  - New unit types
  - Starting perks
